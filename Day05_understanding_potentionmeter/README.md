Day 05 - making a circuit to measure voltage using potentiometer (Arduino Basic)

Objective:

Understanding Potentiometer

Components Used:

- Arduino Uno
- USB Cable
- Potentiometer (10k)
- bread board and jumper wires

What I did:

Write a program to measure the voltage using potentiometer
Problem Faced:
Don't know how potentiometer works how measure Volt

Solution:

- to measure volt use Analog pin and use analogRead command
- to print volt values use Serial.print command
  
What I learned :

- How Serial.begin , Serial.print , Serial.println and analogRead works
- How baurd number is important for showing result
