Day 01 - LED Blink (Arduino basics)

Objective:
Learn how to upload code to Arduino and control the onboard LED.

Components Used :
- Arduino Uno
- USB cable
- Built-in LED (Pin 13)

What I Did : 
Uploaded the Blink example and modified delay values.

Initially confused by board selection and port.

Solution:
Download the driver (CH34), Selected correct COM port and Arduino Uno board.

What I learned :

- How digitalWrite controls output
