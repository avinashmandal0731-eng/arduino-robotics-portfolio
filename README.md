## About this repo

This repository documents my Arduino learning journey following the Paul McWhorter series. Each folder contains one session — the `.ino` sketch, a brief explanation, and wiring notes. The goal is to build a strong embedded systems foundation as part of my Open Doors Engineering & Technology preparation.
